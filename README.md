# fluttest
Run Flutter test on interactive mode :tada:

## Usage

Install:
```sh
npm install @dicodingacademy/fluttest -g
```

for running all test in your project, run this command
```sh
fluttest --all
```

Run single test file? Do this
```sh
fluttest --source file_test.dart
```
